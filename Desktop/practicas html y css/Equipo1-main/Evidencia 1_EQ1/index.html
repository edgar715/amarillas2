¿Qué es el Desarrollo Web?

Es el proceso de creación, construcción y mantenimiento de sitios y aplicaciones web. Implica varios aspectos, como el diseño, la programación, la gestión de bases de datos, y la configuración de servidores, entre otros. El desarrollo web permite a las empresas, individuos o instituciones crear su presencia en línea y proporcionar servicios o información a través de internet.

El primer sitio web del mundo fue creado por Tim Berners-Lee en 1991. Fue diseñado para explicar cómo funciona la World Wide Web: http://info.cern.ch/.

Modelo Cliente-Servidor

Es una arquitectura donde los clientes (dispositivos del usuario) solicitan servicios o recursos al servidor, el cual procesa dichas solicitudes y responde con los resultados. Este modelo es la base de la interacción en la web.

Componentes del Modelo

Cliente

En el contexto de la web, un cliente suele ser un navegador web que solicita páginas HTML, imágenes, etc.

Servidor

Un servidor web entrega las páginas solicitadas al cliente.

Comunicación

Los clientes y servidores se comunican a través de un protocolo de red, siendo HTTP/HTTPS los más comunes. El cliente envía una solicitud al servidor, y este devuelve la respuesta en forma de página web u otros recursos.

Puertos de Comunicación

Un puerto es un punto de acceso en una computadora para la transmisión de datos. Los puertos permiten que los dispositivos se comuniquen a través de la red utilizando diferentes protocolos.

Puertos comunes

  • Puerto 21 - FTP.
  • Puerto 22 - SSH.
  • Puerto 25 - SMTP.
  • Puertos 80 y 443 - HTTP y HTTPS.

Web Server

Un servidor web es un software que entrega contenido web a los usuarios. Gestiona las solicitudes de los clientes y ofrece recursos como páginas HTML, archivos y medios.

Funciones principales

  • Gestión de solicitudes HTTP.
  • Alojamiento de sitios web.
  • Seguridad (SSL/TLS).

Funcionamiento de una Aplicación Web

Una aplicación web es un programa que se ejecuta en un servidor y se accede a través de un navegador web. Este tipo de aplicaciones interactúan con el usuario mediante una interfaz gráfica que se muestra en el navegador.

Componentes

  • Front-end (cliente): HTML, CSS, JavaScript.
  • Back-end (servidor): Lógica, bases de datos, autenticación.
  • API: Comunicación entre aplicaciones.

Despliegue de Contenido Web

Desarrollo

Creación del sitio web o aplicación en un entorno local, donde los desarrolladores escriben código y prueban las funcionalidades antes de lanzarlo a producción.

Pruebas

Verificación de que el sitio web funciona correctamente, detectando errores y corrigiéndolos antes de lanzarlo al público.

Deploy

Transferencia de archivos a un servidor de producción, donde el sitio web estará accesible para los usuarios finales.

Mantenimiento

Actualizaciones y ajustes necesarios para asegurar el funcionamiento continuo del sitio web, incluyendo la resolución de problemas de seguridad y la implementación de nuevas funcionalidades.
