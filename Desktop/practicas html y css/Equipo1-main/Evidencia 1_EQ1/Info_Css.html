¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación visual de un documento HTML. Se encarga de cómo los elementos de una página web deben ser mostrados en términos de diseño, colores, fuentes, márgenes, y mucho más. CSS es esencial para crear sitios web atractivos, responsivos y accesibles.

¿Por qué es importante CSS?

CSS es crucial para el desarrollo web moderno, ya que permite separar el contenido (HTML) de la presentación. Esto facilita el mantenimiento, mejora la accesibilidad y hace que el diseño sea más flexible. A continuación se destacan algunas razones clave:

  • Diseño visual: CSS define cómo se verá un sitio web, permitiendo aplicar colores, tipografías y layouts.
  • Responsividad: Con CSS, es posible crear sitios web que se adapten a diferentes dispositivos (móviles, tabletas, ordenadores).
  • Optimización del código: Al separar el contenido del estilo, el código es más limpio y fácil de mantener.

Ejemplo básico de CSS

A continuación te mostramos un ejemplo simple de cómo usar CSS para cambiar el color y la tipografía de un título en HTML.

HTML:

<h1>Hola Mundo</h1>

CSS:

h1 {
  color: blue;
  font-family: Arial, sans-serif;
}

En este ejemplo, el CSS cambia el color del texto del título <h1> a azul y aplica la fuente Arial.

Conceptos clave de CSS: Selectores y Propiedades

En CSS, los selectores se utilizan para seleccionar los elementos HTML a los que se les aplicará estilo. Las propiedades definen qué estilo se aplicará a esos elementos. A continuación algunos ejemplos comunes:

  • Selector de etiqueta: Aplica estilos a todas las etiquetas de un tipo específico (por ejemplo, h1).
  • Selector de clase: Se aplica a cualquier elemento con un atributo de clase específico. Se denota con un punto . antes del nombre de la clase (por ejemplo, .mi-clase).
  • Selector de ID: Se aplica a un solo elemento con un atributo de ID específico. Se denota con una almohadilla # antes del nombre del ID (por ejemplo, #mi-id).

Ejemplo avanzado de CSS: Diseño de un contenedor

CSS también permite aplicar diseño avanzado a los elementos de una página. A continuación te mostramos cómo se puede diseñar un contenedor con bordes, relleno (padding) y margen.

HTML:

<div class="contenedor">Contenido aquí</div>

CSS:

.contenedor {
  border: 2px solid #555;
  padding: 20px;
  margin: 20px 0;
  background-color: #f4f4f4;
}

Este CSS aplica un borde, relleno y margen al contenedor, creando un bloque bien definido que resalta el contenido.
