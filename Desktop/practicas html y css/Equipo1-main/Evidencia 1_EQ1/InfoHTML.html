¿Qué es HTML?

Hyper Text Markup Language por sus siglas en inglés, (Lenguaje de marcado de hipertexto) es un lenguaje de etiquetado cuyo objetivo es realizar estructuras para el texto en la web basándose en hipertexto. Siendo la versión 5 la más actual donde introduce nuevas etiquetas, elementos y APIs que mejoran la funcionalidad y la interoperabilidad de las aplicaciones web. Algunas de sus características son:

  • Utiliza etiquetas
  • Se realiza mediante texto plano (símbolos, letras, números)
  • Puede crearse desde cualquier procesador de texto.
  • Se debe generar un archivo con extensión .html.
  • Es interpretado por los navegadores.
  • Cuenta con una estructura de etiquetas pre definidas.

    • ¿Por qué es importante HTML?

    Es importante porque es el lenguaje base que estructura el contenido de las páginas web como:

    • Estructura de la web
    • Compatibilidad universal
    • Accesibilidad
    • Optimización para motores de búsqueda
    • Interacción con otros lenguajes

    Estructura de HTML

    A continuación te mostramos la estructura básica a usar para hacer una página de HTML.

    HTML:

    <!DOCTYPE HTML>
    <html>
    <head>
      <title></title>
    </head>
    <body>
    </body>
    </html>
    <!DOCTYPE HTML>

    Especifica el tipo de documento y la versión de HTML utilizada.

    Etiquetas más utilizadas


      <html>: Indica el inicio y el fin de una estructura html.

      <head>: Es la parte superior o cabecera de una página web.

      <title>: Hace referencia a los títulos y sus características visuales, suele ir dentro del head.

      <br>: Crea un salto de línea entre nuestros elementos.

      <p>: Crea un salto de línea espaciado entre los elementos (párrafo).

      <body>: Dentro de esta etiqueta acomodamos todo el contenido de la página, textos, tablas, imágenes y todo lo que necesitemos para la misma.

      <h>: Del 1 al 6 en orden de importancia, estas etiquetas se utilizan para encabezados como por ejemplo <h1> <h2>

      <a>: Define un hipervínculo

      <li>: Define un elemento dentro de una lista (listitem)

      <ol>: Define un conjunto de elementos ordenados

      <ul>: Define un conjunto de elementos desordenados

      <table>: La versión genérica de tablas para HTML, en si debe contener al menos una etiqueta <tr> (table row, una fila) y un <td> (table data, elemento dentro de una fila, en esencia, una columna)

      <img>: Inserta imágenes en una página web.

      <fildset>: Agrupa elementos dentro de un formulario.

      <legend>: Se emplea dentro de un <fildset> para proporcionar una etiqueta o título.

      <div>: Es un contenedor genérico de bloque que se utiliza para agrupar y estilizar elementos mediante CSS.

      <blockquote>: Se utiliza para indicar una cita de bloque, es decir, una cita extensa de otro texto.

      <article>: Se emplea para definir un contenido independiente y autocontenido dentro de un documento, como una entrada de blog, un artículo de noticias, o una sección de una revista en línea.

    Ejemplo para página HTML

    Código:

    <!DOCTYPE HTML>
    <htm lang="es">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Página Web </title>
    </head>
    <body>
      <h1>Bienvenido a Mi Página Web</h1>
      <p>Ejemplo de una página web súper sencilla sin estilos CSS adicionales. Aquí puedes ver cómo se usan las etiquetas HTML básicas</p>

      <img src="https://via.placeholder.com/400" alt="Imagen de ejemplo">

      <p>Para más información, visita <a href="https://www.example.com" target="_blank"> mi sitio web favorito</a> </p>
    </body>
    </html>

    Formulario

    Input tipo text

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene un cuadro de texto donde el usuario puede ingresar información, con una sugerencia de "Nombre".

    <td>TEXT</td>
    <td><input type="text" name="txttext" placeholder="Nombre"></td>

    Input tipo number

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene un campo de entrada solo permite números.

    <td>NUMBER</td>
    <td><input type="number" name="txtNUMBER"></td>

    Input tipo pasword

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiendra caracteres que el usuario ingrese y estarán ocultos (normalmente se muestran como puntos o asteriscos) para mayor seguridad..

    <td>PASSWORD</td>
    <td><input type="password" name="txtpassword"></td>

    Input tipo radio

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene dos botones de opción (radio buttons) y etiquetas de opción..

    <td>RADIOBUTTON</td>
    <td><input type="radio" name="rdRadio1" value="opcion1">OPCIÓN1<br>
    <input type="radio" name="rdRadio2" value="opcion2"">OPCIÓN2</td>

    Input tipo checkbox

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene una casilla de verificación que el usuario puede marcar o desmarcar..

    <td>CHECKBOX</td>
    <td><input type="checkbox" name="chkcheckbox">Aceptar></td>

    Input tipo date

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene un campo de entrada que permite al usuario seleccionar una fecha..

    <td>DATE</td>
    <td><input type="date" name="txtDATE"></td>

    Input tipo time

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda contiene un campo de entrada que permite al usuario seleccionar una hora, dependiendo del navegador y la configuración.

    <td>TIME</td>
    <td><input type="time" name="txtTIME"></td>

    Input tipo file

    Crea una fila con dos celdas. La primera celda contiene un texto estático, mientras que la segunda celda permite al usuario seleccionar un archivo de su dispositivo. La configuración con el atributo accept limita la selección a archivos PDF.

    <td>FILE</td>
    <td><input type="file" name="txtFILE" accept=".pdf"></td>
